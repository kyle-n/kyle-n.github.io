Dragon Collector

Dragon Collector is a website for collecting and generating pixel art dragons.

Interesting problem: I had to figure out how to generate random shades of a specific color. Building my color generation algorithm was a fascinating look into RGB and hex codes. I loved solving a problem that probably no one else has had to solve.

Tools: Angular 7, MongoDB, Mongoose, Express.js, Node, Angular Material, Passport.js, JWTs, bcrypt, RxJS

Backlogged

Backlogged is a simple and clean Letterboxd for video games. Powered by the Giant Bomb API, it lets users track games they've played with lists and ratings.

Interesting problem: I created an API route that would convert a list of game titles to an image for sharing on social media (example). I loved figuring out how to lay out a block of text when you don't know how many items there are or how long those items will be. It was a great challenge to create the rules so that text block wouldn't just work, but look appealing (e.g. roughly equal height and width).

Tools: Angular 6, MongoDB, Mongoose, Express.js, Node, Passport.js, request-promise, JWTs

What Should I Play Next?

What Should I Play Next is a client-side Angular app for finding new games to play based on their ratings. It uses the Giant Bomb API as well.

Interesting problem: I had to understand the minute details of the API in order to get the data I wanted (review scores sorted by platform), which was not supported. It was a great learning experience to figure out how to take the data I could get and make it into the data I couldn't get (repeated requests to get reviews while filtering by platform).

Tools: Angular 6, Express.js, Node
